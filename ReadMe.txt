Файл описания работы приемной станции

После подключения питания на дисплее выводится название устройства
После этого начинает работать радиомодуль и выводить на дисплее принятые значения цветов

RED - значение красного канала
GREEN - значение зеленого канала
BLUE - значение синего канала
SINGLE - значение одноцветного канала
